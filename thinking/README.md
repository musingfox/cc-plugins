# Thinking

Thinking frameworks for better decision-making: scenario-driven design, dimensional analysis, and explicit trade-offs.

## Skills

### scenario-thinking

Activated when designing features, choosing between implementation approaches, making architecture decisions, or any situation with multiple viable options.

**What it does:**
1. Asks about future usage scenarios before proposing solutions
2. Analyzes relevant dimensions (performance, cost, complexity, etc.)
3. Proposes solutions with explicit trade-offs
4. Provides comparison tables for multiple options

## Installation

```bash
/plugin install thinking
```
